# M3U8 视频链接检测助手

一款轻量级的油猴脚本，用于自动检测网页中的 M3U8 视频流链接。

[![安装脚本](https://img.shields.io/badge/安装脚本-点击安装-brightgreen?style=for-the-badge&logo=tampermonkey)](https://github.com/MissChina/showmv/raw/main/M3U8视频链接检测助手.user.js)

> 点击上方按钮直接安装（需先安装油猴脚本管理器）

## 功能特点

- **自动检测**：自动拦截并识别页面中的 M3U8 视频链接
- **智能验证**：通过多重验证机制确认链接有效性
- **一键复制**：检测到的链接支持一键复制到剪贴板
- **实时日志**：提供详细的检测日志，方便调试和追踪
- **可拖拽面板**：浮动面板支持自由拖动，不影响正常浏览

## 安装方法

### 前置要求

请先安装以下任一油猴脚本管理器：

- [Tampermonkey](https://www.tampermonkey.net/)（推荐）
- [Violentmonkey](https://violentmonkey.github.io/)
- [Greasemonkey](https://www.greasespot.net/)

### 安装步骤

1. 安装并启用油猴脚本管理器
2. 点击脚本文件 `M3U8视频链接检测助手.user.js`
3. 在弹出的安装页面点击「安装」按钮
4. 安装完成后刷新目标页面即可使用

### 更新方法

- **自动更新**：脚本管理器会定期检查更新
- **手动更新**：在脚本管理器中删除旧版本，重新安装新版本

## 使用说明

1. 安装脚本后，访问任意视频网站
2. 当检测到 M3U8 链接时，页面右上角会自动弹出检测面板
3. 在「链接」标签页可查看所有检测到的 M3U8 地址
4. 点击「复制链接」按钮可将地址复制到剪贴板
5. 在「日志」标签页可查看详细的检测过程

## 技术原理

本脚本通过以下方式检测 M3U8 链接：

- Hook `fetch` API 拦截网络请求
- Hook `XMLHttpRequest` 拦截 AJAX 请求
- 使用 `PerformanceObserver` 监听资源加载
- 通过 `.ts` 分片请求辅助验证真实地址
